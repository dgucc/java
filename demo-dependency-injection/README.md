# Dependency Injection
How to apply Loose Coupling instanciation  

## Using Properties example
`mvn exec:java -Dexec.mainClass="AppDynamicInstanceProps"`  

## Using Spring Annotation example
`mvn exec:java -Dexec.mainClass="AppSpringAnnotation"`   

## Using Spring config.xml example
`mvn exec:java -Dexec.mainClass="AppSpringXML"`
